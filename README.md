# Weather App

## Description:

The Weather App is a simple web application that allows you to get information about the current temperature of a city by entering its ZIP code. By providing the ZIP code, the application returns the city name, current temperature, and the date when the weather data was collected.

## Usage Instructions:

1. **Enter ZIP code:** On the main page of the application, you'll find an input field where you can enter the ZIP code of the city you want to check.
2. **Enter you feelings:** After entering the ZIP code, you can enter how are you feeling.
3. **View results:** Once the request is submitted, the application will display the city name, current temperature, and the date when the weather data was collected plus your feel!.

## Requirements:

- Updated web browser (Google Chrome, Mozilla Firefox, Safari, etc. recommended).
- Internet connection to fetch real-time weather data.

## Technologies Used:

- HTML: For the structure and content of the web page.
- CSS: For the design and visual presentation of the application.
- JavaScript: For user interaction and application logic.
- Weather forecast API: OpenWeatherMap API to retrieve weather data based on the entered ZIP code.

## Setup:

No special setup is required to use the Weather App. Simply open the HTML file in your web browser and start using the application.
